WeatherSky is a modern, high-end weather application that provides real-time weather information for any city or country in the world. It features a stunning glassmorphic UI with dynamic background animations and instant location search.

## 🔗 Live Demo
**View the live application here:** [https://zsasannia70-png.github.io/weather-sky/](https://zsasannia70-png.github.io/weather-sky/)


## ✨ Features

- **Real-Time Data**: Get current temperature, humidity, wind speed, and cloudiness.
- **Global Search**: Search for any location worldwide using integrated geocoding.
- **Quick Selection**: Fast access to weather data for major global cities (London, New York, Tokyo, Paris).
- **Premium Design**: 
    - Glassmorphism effects
    - Dynamic, animated blob backgrounds
    - Responsive layout for mobile and desktop
    - High-quality typography (Outfit font)
- **Zero Setup**: Works immediately out of the box using the Open-Meteo API (no API key required).

## 🚀 Getting Started

The application is built with vanilla HTML, CSS, and JavaScript, making it extremely easy to run.

### Prerequisites

- A modern web browser.
- A local web server (optional, but recommended for the best experience).

### Running the App

1.  Clone or download the repository.
2.  Open `index.html` in your browser.
3.  Alternatively, use a local server:
    ```bash
    npx serve
    ```
4.  Navigate to `http://localhost:3000` (or the port specified by your server).

## 🛠️ Built With

- **HTML5**: Semantic structure.
- **CSS3**: Premium styling, glassmorphism, and animations.
- **JavaScript (ES6+)**: Fetch API, Geocoding, and DOM manipulation.
- **Open-Meteo API**: Free, open-source weather data.
- **Google Fonts**: Outfit font for a modern look.
